# AndriodApkBuilder

## 介绍

本项目提供构建开源安卓应用的 GitHub Actions 工作流集合。

## 支持应用

### BugVideo

[BV (Bug Video)](https://github.com/aaa1115910/bv) 是一款十分优秀的 [哔哩哔哩](https://www.bilibili.com/) 的第三方 Android TV 应用，使用 Jetpack Compose 开发，支持 Android 5.0+ 设备。
